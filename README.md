# FoodCulture - Full Stack Recipe Application

## Overview

FoodCulture is a full-stack recipe management web application developed using React, Node.js, Express.js, and MongoDB. The application allows users to explore recipes, manage recipe data, and interact with a responsive web interface connected to a backend REST API.

This project was developed to practice and improve full-stack web development skills, including frontend development, backend API creation, database integration, and user authentication.

## Features

### User Features
- User registration and login
- Password encryption
- JWT-based authentication
- Secure user access

### Recipe Management
- View recipes
- Add new recipes
- Update existing recipes
- Delete recipes
- Store and retrieve recipe data from MongoDB

### Frontend Features
- Responsive user interface
- Recipe browsing layout
- Food culture section with different cuisines
- Interactive forms
- API integration using Axios

## Technologies Used

### Frontend
- React.js
- HTML5

- Axios

### Backend
- Node.js
